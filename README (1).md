# MHT-CET Preparation Bot

A Telegram bot with 183 practice questions for MHT-CET exam preparation covering Physics, Chemistry, and Mathematics.

## Features
- 183 Real MHT-CET style questions
- 3 Difficulty levels (Easy, Moderate, Hard)
- Progress tracking
- Performance analytics
- Instant feedback with explanations

## Setup Instructions

1. Get your bot token from [@BotFather](https://t.me/BotFather) on Telegram
2. Replace `BOT_TOKEN` in `READY_TO_RUN_BOT.py` with your token
3. Deploy on Render.com or any Python hosting platform

## Deployment

### On Render.com:
1. Fork this repository
2. Sign up on [render.com](https://render.com)
3. Create new Web Service
4. Connect your GitHub repo
5. Set Start Command: `python READY_TO_RUN_BOT.py`
6. Deploy!

## Requirements
- Python 3.7+
- pyTelegramBotAPI
